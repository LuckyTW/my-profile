## 3. 기술 스택 & 구현 조건
- HTML, CSS, JavaScript만 사용
- Tailwind CSS는 **CDN 방식**으로 적용
- React, Vue 등 외부 프레임워크 사용 금지
- 모든 코드는 **하나의 HTML 파일**에서 바로 실행 가능하도록 작성
- 이미지 사용 시:
  - 외부 이미지 대신 그라데이션, 패턴, 이니셜 아바타 등으로 대체

## 4. 디자인 요구사항
- 모던하고 깔끔한 디자인
- 충분한 여백과 명확한 정보 계층
- 모바일 / 태블릿 / 데스크탑 반응형 대응
- 과하지 않은 애니메이션 (hover, transition, smooth scroll)
- 가능하다면 **다크모드 토글 제공**
  - localStorage에 테마 상태 저장

---

## 5. 코드 품질 기준
- 시맨틱 HTML 태그 사용 (header, main, section, footer 등)
- Tailwind 클래스는 섹션 단위로 정리하여 가독성 유지
- 각 섹션에 주석 추가 (추후 수정 용이)
- 접근성 고려:
  - focus 스타일
  - aria-label 적절히 사용
- UX:
  - 상단 네비게이션 sticky
  - 스크롤 시 현재 섹션 active 표시
  - 부드러운 앵커 이동
